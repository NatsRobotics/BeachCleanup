# BeachCleanup

Open index.html to run, or go to https://natsrobotics.github.io/BeachCleanup/ (May be down for a few minutes during updates)<br>
Controls:<br>
-WASD to move<br>
-Shift to sprint<br>
-Click on garbage to collect<br>
-p to pause/buy upgrades
